# Skellington

World's Greatest Sinatra Skeleton.

DataMapper (ORM), RSpec & Webrat (Testing), Bootstrap + HAML + AssetPack
(Composing), and Bundler (Bundling).

## Setup

Tested on Ruby 2.0.0-p247

```
bundle install
bundle exec rackup
```
