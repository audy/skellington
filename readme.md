# Skellington

World's Greatest Sinatra Skeleton
Inspired by Padrino, Monk, Lolwut, Klee, and others.

- DataMapper
- RSpec
- Webrat
- Twitter Bootstrap
- HAML
- AssetPack
- Bundler

Comes with a simple User model, lovingly ripped from Padrino.

## Setup

Tested on Ruby 2.0.0-p247

```
bundle install
bundle exec rackup
```
